# 简介

这个项目基于 Arch linux，旨在学习Liunx系统相关知识，项目宗旨：
1. 系统稳定第一位（语言使用优先级：shell、C、C++、glib、Qt）
2. 尽可能花里胡哨
3. 项目基本fork自开源社区

## 包状态

|------|----|
|||
